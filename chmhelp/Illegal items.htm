Illegal items

In PC Simulator, normally you can only get items through the Daily Market, Print Expert, and Store.
However, there are a few items that are normally unobtainable.

Hidden, but not illegal items

You can go to the Silk Road website to buy these stuff:

PUOIO Phone

It's just a phone. An iPhone.

Super Glue

This item make it so that it stays plugged in invisibly even when the component is removed from the PC.
All you have to do is to make it touch a PC component and it will add glue to it. And bam! It remains recognized all the time!

Illegal items

Projector

An actual projector and it works as a display! All you have to do is to set this as the monitor for your PC.

RTX4080Ti

This was originally going to be implemented as a normal item, but it was too big to fit into computers and thus was made an illegal item.
Plus, it is very bad at benchmarks and mining.
