Welcome to PC Simulator Save Editor Help. I'll guide you how to use the program.

First step: Taking a look at the program



The left textbox is where you put your encrypted/decrypted texts to decrypt/encrypt.
The right textbox is where the encrypted/decrypted files get decrypted/encrypted.

The "Select Save File" button is where you select the files to decrypt/encrypt if you dont want to copy-paste.
The "Save to..." button is where after you are done with the output text, you can save it to a save file. Do note here, that it is automatically decrypted/encrypted.

The "Copy to clipboard" button copies the output text to the clipboard.
The "Guide" button is self-explanitory.
